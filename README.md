# Tunable Artificial Atoms – Quantum Engineering Dataset

**Manuscript**: "Tunable Artificial Atoms: Engineered Quantum Building Blocks for Scalable Processors and Unhackable Networks"  
**Author**: Umair Abbas Siddiquie (ORCID: 0009-0008-3968-2252)  
**License**: MIT  

## Contents
- `/data/raw/` – Cryogenic measurement logs (HDF5 format)
- `/data/processed/` – Coherence time, fidelity, crosstalk matrices (CSV)
- `/simulations/` – Circuit-QED models (QuTiP + Python)
- `/designs/` – Fabrication layouts (GDSII + PDF schematics)
- `/notebooks/` – Jupyter analysis scripts (reproduce figures)
